# Origin 与 Upsream 的区别

- ### Origin

  指的是源仓库。就是你远程本人创建的仓库。

- ### Upsream

  指的是上游仓库。比如你在github 上fork了 别人的开源项目，开源项目就复制到你的github仓库上。那么别人的开源项目就是upstream, 你自己GitHub仓库那一份就是origin
